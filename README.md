# GitHub API Candidate Search App

## Overview
This application allows users to search for GitHub candidates based on specific criteria using the GitHub API.

## Features
- Search for candidates by username
- Filter candidates by location, repositories, and followers
- View detailed candidate profiles

## Installation
1. Clone the repository:
  ```sh
  git clone https://github.com/yourusername/githubapi-candidate-search.git
  ```
2. Navigate to the project directory:
  ```sh
  cd githubapi-candidate-search
  ```
3. Install the required dependencies:
  ```sh
  npm install  
  npm run build
  ```

## Usage
1. Start the application:
  ```sh
  npm run dev
  ```
2. Open your browser and navigate to `http://localhost:3000`.

## Configuration
- Create a `.env` file in the root directory and add your GitHub API token:
  ```
  GITHUB_API_TOKEN=your_github_api_token
  ```
